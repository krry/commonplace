# UXioms spec

I want to create a site where the UX-minded can discuss, discover, and define the axioms of UX. This field is long overdue for a clean, true set of operating principles. The principles should be described and then exemplified in context.

So we'll need many pages, and an index/table of contents. We'll want to collect input from visitors in the form of responses to surveys, and then generate future views based on collected data. L
