# Mac OSX Dev Environment Setup and Toolset

## Install
### The Basics
* [Xcode](//itunes.apple.com/us/app/xcode/id497799835?ls=1&mt=12)                                                
* [Sublime Text](//sublimetext.com)
* [Atom](https://atom.io/)

### CLI
* [git](//git-scm.com/) - get git, it comes with Xcode* [node.js](//nodejs.org) - shortest path to a web prototype
* [nvm](//nvm.io) - Node Version Manager
* [rvm](//rvm.io) - Ruby Version Manager
* [Heroku Toolbelt](https://toolbelt.heroku.com/)

### Package Managers
* [npm](//npmjs.org) - should come with Node `npm install npm -g`
* [RubyGems](//rubygems.org) - `gem update --system`
* [Bower](//bower.io) - `npm install -g bower` - front-end package manager for things like Angular, Backbone, jQuery

### Global packages
* [Gulp](//gulpjs.com/) - `npm install -g gulp` - super app builder
* [Ender](//enderjs.com/) - `npm install -g ender` - open module JS framework
* [Browserify](//browserify.org/) - `npm install -g browserify` - client-side JS manager
* [Sass](//sass-lang.com/) [for Gulp](//www.npmjs.org/package/gulp-sass) - `npm install -g gulp-sass` - premier CSS pre-compiler lang

## Configure
* .dotfiles - Canseco: A Juiced-up [Bash Profile](//gist.github.com/krry/9263628)
* [Preferences.sublime-settings](//gist.github.com/krry/37dda3478008b4cca19f)
* [OSX for Hackers](//gist.github.com/krry/9263570)
* [github](//github.com) - on the command line, use `git config` to set user.name, user.email
    git config --global user.name "Firstname Last"
    git config --global user.email "name@domain.com"
* git - add RSA key
    cd ~/.ssh
    ssh-keygen -t rsa -C "name@domain.com"
    eval "$(ssh-agent -s)"
    ssh-add ~/.ssh/id_rsa
    pbcopy < ~/.ssh/id_rsa.pub
    ssh -T git@github.com
* git - [.gitignore_global](//gist.github.com/krry/ac01059cb5683453a0b8)
		git config --global core.excludesfile ~/.gitignore_global
		cd ~
		git clone https://gist.github.com/krry/ac01059cb5683453a0b8
* Heroku - add RSA key - `heroku keys:add`
