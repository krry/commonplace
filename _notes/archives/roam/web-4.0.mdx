# Web 4.0- the next next gen of the internet
    - People and things transact with trust via blockchain, an open, secure distributed ledger
- [[Tenets]] of Web 4.0
    - [Attention]([[attention]]) (as focused time) is value.
    - [[Reputation]] drives attention.
    - [[Trust]] underpins reputation.
    - [[Feedback loops]] (often transactions or gifts) build trust.
    - [[Interface]] enables feedback and [[transactions]].
    - [[blockchain]] effectively cheapens trust.
