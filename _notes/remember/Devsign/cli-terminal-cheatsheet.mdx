# Terminal Cheatsheet

- `htop` - to monitor processes and processing load on CPUs, Memory, Swap drive, Uptime and such
- `refish` - to reload the fish shell and source its config again
  - `rebash` and `rezsh` similarly
- `ccd` - to jump to the Code dir
- `brin` and `brun` and `bruu` to install and uninstall and update/upgrade `brew`
