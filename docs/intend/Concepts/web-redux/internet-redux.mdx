# What would the Internet comprise if we invented it today?

1. **Communicator** - instantly message anyone you know, reach them anywhere, record and archive your conversations
2. **Moment capture** - in motion or still, with sound or not, condensed media or RAW
3. **Data collection** - taxonomies
4. **Storage** - aggregation, scrubbing
5. **Data analysis** - crunching
6. **Data verification** - knowledge refining
7. **Library** - pan-encyclopedia, query for any historical information, find suggested/related info
8. **Entertainment** - distribution/streaming
